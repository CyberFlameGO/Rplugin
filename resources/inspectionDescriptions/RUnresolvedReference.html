Reports unresolved symbol and function references.

Example:


my_fahrenheit_to_celsius(40)

The my_fahrenheit_to_celsius function is not defined, so the IDE shows a warning. You should add the missing declaration.

Possible solution: 


my_fahrenheit_to_celsius  <- function(f_temp) {
  return((f_temp - 32) * 5 / 9)
}

my_fahrenheit_to_celsius(40)
