Filter
  • value: keep rows that contain value as a substring; wildcard characters (*, ?) are allowed
  • =value: keep rows that are equal to value
  • !value: keep rows that are not equal to value
  • <value: keep rows that are less than value
  • <=value: keep rows that are less than or equal to value
  • >value: keep rows that are greater than value
  • >=value: keep rows that are greater or equal than value
  • ~~regex: keep rows that match the given regular expression
  • ~pattern: keep rows that match the given pattern with wildcards (*, ?)
  • !~pattern: keep rows that does not match the given pattern
  • _: keep rows with NA
  • !_: keep rows without NA
